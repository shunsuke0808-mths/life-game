Firebase Setup

⚙️ 初期設定ガイド
① Firebaseプロジェクト作成
console.firebase.google.com にアクセス→「プロジェクトを追加」
② Realtime Database を有効化
左メニュー「構築」→「Realtime Database」→「データベースを作成」→ テストモードで開始
③ ウェブアプリを追加
プロジェクト設定（歯車）→「アプリを追加」→ </> ウェブ → 設定オブジェクトをコピー
④ このファイルを編集
このHTMLファイルの上部にある firebaseConfig 部分を、コピーした値に書き換える
⑤ セキュリティルール（任意）
Realtime Database → ルール → 以下に変更:
{"rules":{".read":true,".write":true}}

設定が完了したら以下のボタンを押してください
